How to make a custom Version Profile: (Example with OptiFine)
  1. Select the Version of your custom jar
  2. Don't select anything in the selection menu beneath
  3. Start the profile and close again after successful start
  4. Press "Show Folder" Button on profile page
  5. Remember or copy folder location that opened in explorer
  6. Run OptiFine installer and install into the directory you copied the path of (Close launcher beforehand)
  7. Edit the Profile again and select your JAR
  8. Save profile and start Minecraft
